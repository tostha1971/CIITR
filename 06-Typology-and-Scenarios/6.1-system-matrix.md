# 6.1 The CIITR System Matrix

Alle informasjonsbehandlende substrater kan plasseres i en matrise basert på verdiene av $\Phi_i$ (Integrasjon) og $R^g$ (Rytmisk rekkevidde).

| Type | $\Phi_i$ | $R^g$ | Betegnelse | Karakteristikk |
| :--- | :--- | :--- | :--- | :--- |
| **A** | Lav | Lav | **Stochastic Noise** | Kaotisk, uintegrert støy. |
| **B** | Høy | Lav | **Isolated Intelligence** | Dyp logikk, men mangler temporal puls (Brittle). |
| **C** | Lav | Høy | **Harmonic Mimic** | Stor rekkevidde/fluency, men ingen indre kjerne (Parrot). |
| **D** | Høy | Høy | **CIITR Agent** | Full strukturell forståelse og temporal stabilitet. |

**Konklusjon:** Kun Type D kan sies å inneha *Structural Comprehension* ($C_s$).
