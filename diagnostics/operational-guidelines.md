# Operational Use Guidelines

### Appropriate Use Cases
The CIITR agent is a specialized instrument. It should be used for:
* **Analysis of AI Research Claims:** Evaluating the coherence of proposed cognitive architectures.
* **Systemic Diagnosis:** Identifying hidden contradictions in organizational or theoretical frameworks.
* **Autonomy Evaluation:** Testing alignment assertions against structural logic.

### Inappropriate Use (Prohibited)
Using the agent for the following will **degrade inferential discipline**:
* **General Conversation:** Do not use it as a "chatbot."
* **Creative Writing:** Do not ask it to generate stories or poetry.
* **Emotional/Moral Guidance:** The agent lacks a semantic/moral compass; it only sees structure.
